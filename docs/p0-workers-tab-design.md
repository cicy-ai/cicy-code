# Workers Tab — 重新设计

## 目标
一屏看到所有 Worker 的工作状态，同时适配小屏（~320px）和超大屏（2K+）。

## 核心思路
- 用 **网格布局** 展示所有 worker 的 mini terminal
- 每个 worker 是一个 **卡片**，内嵌 ttyd iframe
- 网格列数根据容器宽度自适应
- 点击卡片可以放大（全屏 modal）

## 卡片设计

```
┌─────────────────────────────────┐
│ 🔧 worker-name   claude-haiku  │  ← 标题 + model (truncate)
│ ● idle                    0.2$ │  ← 状态 + 成本 (未来)
├─────────────────────────────────┤
│                                 │
│         ttyd iframe             │  ← 实时终端 (pointer-events: none)
│         (只读预览)               │     点击卡片才激活交互
│                                 │
├─────────────────────────────────┤
│ [↗ Open]  [⏸ Pause]  [✕ Unbind]│  ← 操作栏
└─────────────────────────────────┘

状态边框颜色:
  thinking → 左边框 2px 黄色 (#d29922)
  idle     → 左边框 2px 绿色 (#3fb950)
  offline  → 左边框 2px 红色 (#f85149) + 内容灰度
```

## 响应式规则

三种模式，用 ResizeObserver 监听容器宽度切换。

### 小屏 (< 480px) — 强制单列，拉伸
```
┌─────────────────────────┐
│ 🔧 w-1  ● idle          │  卡片 width: 100%
│                         │  WebFrame 高度自由拉伸
│   WebFrame (flex-1)     │  (占满剩余空间)
│                         │
│ [↗] [✕]                │
├─────────────────────────┤
│ 🔧 w-2  ● think         │
│                         │
│   WebFrame (flex-1)     │
│                         │
│ [↗] [✕]                │
└─────────────────────────┘
```
- 单列，卡片撑满宽度
- WebFrame 高度用 `flex: 1` 拉伸，每张卡片平分容器高度
- 纵向滚动

### 中屏 (480-960px) — 可一列可两列
```
一列模式 (worker ≤ 2): 拉伸高度
┌─────────────────────────────┐
│ 🔧 w-1  ● idle              │  width: 100%
│                             │  WebFrame 高度 flex-1 拉伸
│   WebFrame (拉伸)           │  平分容器高度
│                             │
│ [↗] [✕]                    │
├─────────────────────────────┤
│ 🔧 w-2  ● think             │
│                             │
│   WebFrame (拉伸)           │
│                             │
│ [↗] [✕]                    │
└─────────────────────────────┘

两列模式 (worker ≥ 3): 固定宽高，同大屏
┌──────────────┐ ┌──────────────┐
│ 🔧 w-1 idle  │ │ 🔧 w-2 think │  固定 420×300
│ WebFrame     │ │ WebFrame     │  不拉伸
│ [↗] [✕]     │ │ [↗] [✕]     │  flexbox wrap
├──────────────┤ ├──────────────┤
│ 🔧 w-3 off   │ │              │
│ WebFrame     │ │              │
│ [↗] [✕]     │ │              │
└──────────────┘ └──────────────┘
```
- worker ≤ 2 时单列，高度拉伸
- worker ≥ 3 时两列，固定宽高（同大屏逻辑），纵向滚动

### 大屏 (960px+) — 固定宽高，flexbox wrap
```
┌────────────────┐ ┌────────────────┐ ┌────────────────┐
│ 🔧 w-1  idle   │ │ 🔧 w-2  think  │ │ 🔧 w-3  idle  │
│ WebFrame       │ │ WebFrame       │ │ WebFrame       │
│ 420×300 固定    │ │ (黄色边框)      │ │                │
│ [↗] [✕]       │ │ [↗] [✕]       │ │ [↗] [✕]       │
└────────────────┘ └────────────────┘ └────────────────┘
┌────────────────┐ ┌────────────────┐
│ 🔧 w-4  off    │ │ 🔧 w-5  idle   │
│ WebFrame       │ │ WebFrame       │
│                │ │                │
│ [↗] [✕]       │ │ [↗] [✕]       │
└────────────────┘ └────────────────┘
```
- 卡片固定 420px 宽，300px 高
- flexbox wrap，从左到右，自动换行
- 不拉伸，左对齐
- 960→2张, 1400→3张, 1800→4张, 2500→5张...

### CSS 概要

```css
.workers-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 8px;
  padding: 8px;
  overflow-y: auto;
  height: 100%;
}

/* 大屏 + 中屏两列: 固定宽高 */
.worker-card {
  width: 420px;
  height: 300px;
  flex-shrink: 0;
  flex-grow: 0;
}

/* 小屏 / 中屏单列: 撑满拉伸 */
.workers-grid.stretch .worker-card {
  width: 100%;
  flex: 1 1 100%;
  min-height: 200px;
  height: auto;
}
```

### 模式判断逻辑

```ts
if (containerW < 480) mode = 'stretch'                    // 小屏: 强制单列拉伸
else if (containerW < 960 && agents.length <= 2) mode = 'stretch'  // 中屏少量: 单列拉伸
else mode = 'fixed'                                       // 中屏多量 + 大屏: 固定 420×300 wrap
```

## 与现有设计的区别

| 现有 | 新设计 |
|------|--------|
| 列表 + 底部单终端 | 网格 + 每个卡片内嵌终端 |
| 一次只看一个 worker | 同时看所有 worker |
| 上下分割拖拽 | CSS Grid 自适应 |
| 点击切换终端 | 点击放大，ESC 关闭 |
